配置ssh与网络

打开ssh

由于没有用到显示器，所以打开ssh连接是必要的。
打开ssh方法：在树莓派系统磁盘根目录添加文件ssh（无需后缀）。

网络配置

要连接ssh，需要设备位于同一局域网，这里讲如何配置无线连接。

在树莓派系统磁盘根目录添加wpa_supplicant.conf文件，编辑内容：

··· bash
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
network={
ssid=”你的WIFI名”
psk=”WIFI密码”
}
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
network={
ssid=”你的WIFI名”
psk=”WIFI密码”
}
···

这样就可以直接开机连上指定WiFi了。
